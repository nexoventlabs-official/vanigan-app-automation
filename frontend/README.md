# Vanigan — Admin Frontend

Vite + React + Tailwind admin console for the Vanigan WhatsApp directory.

## Setup

```bash
cp .env.example .env       # set VITE_API_URL if your backend isn't on :5050
npm install
npm run dev                # http://localhost:5173
```

Sign in with the credentials seeded in `backend/.env`
(default `admin` / `admin@123`).
